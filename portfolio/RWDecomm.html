College Bookstore E-Commerce Templates

Jared Headrick
  • View My Portfolio

Problem

An E-Commerce solution for college bookstores with inferior performance on mobile devices. The mobile site was separate from the desktop site and had many implementation issues, which caused lack of interest from stores to purchase the solution and extremely low conversion rates for those who did.

Objective

Modify existing front-end solution to create 4 responsive template options that can be implemented quickly and easily by a support team upon each store's request and maintained by store personnel.

Toolbox

HTML, CSS3, Responsive Web Design, JavaScript (jQuery)

My Role

Team leader, project manager, front-end developer, and tester.

End Result

Project completed on time and sold templates at 10x the rate of previous mobile solution.
